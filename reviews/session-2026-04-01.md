## What was done

### Language features
- Language `[[<-` assignment via `Language::set_element()` + `rvalue_to_expr()`
- `<<-` at global scope: `set_super()` short-circuits at global env
- S3 dispatch for binary operators (`|`, `+`, `==`, etc.) via `try_s3_binary_dispatch()`
- List comparison operators (element-wise `==`/`!=` by coercing to character)

### Package system
- `library(base)`/`library(stats)` etc. treated as built-in no-ops
- `character.only=TRUE` evaluates the argument (not just literal TRUE)
- Base packages register synthetic `LoadedNamespace` with `export_all()`
- `packageDescription()` uses shared `is_base_package()` check
- `stats4` and `translations` added to base package list
- `vignetteEngine()` returns empty list in query mode

### Builtins
- `I()` (AsIs) ÔÇö adds class "AsIs" to objects
- `args()` returns a proper closure with NULL body (fixes memoise pattern)
- `vapply` named args via `CallArgs` (X=, FUN=, FUN.VALUE=)
- `sub`/`gsub`/`grep`/`grepl` on NULL return empty vectors

### C API
- `Rconn` struct: complete field layout + `R_CONNECTIONS_VERSION=1`
- `Rf_isPairList()` inline function
- `R_check_class_etc()` with implicit class support (matrix, array, etc.)
- `R_new_custom_connection()` + `R_GetConnection()` runtime stubs

### Grid graphics
- Grob constructors: textGrob, rectGrob, linesGrob, pointsGrob, circleGrob, polygonGrob, segmentsGrob, nullGrob, gTree, gList
- `just` normalization (string Ôćĺ numeric pair)
- `BinaryOp::is_comparison()` method
